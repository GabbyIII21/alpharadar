# Design System

## Theme

Alpha Radar uses a premium dark trading-terminal theme with glassmorphism, restrained neon accents, and high-contrast typography.

## Visual Principles

- Dark background by default
- Neon blue for primary intelligence and active states
- Purple for AI and narrative highlights
- Green for bullish confirmation
- Red for bearish risk
- Amber for warning and high-priority alerts
- Dense but readable dashboard composition
- No stock images
- Custom radar visualization for brand identity

## Components

- Radar score cards
- Event cards
- Narrative cards
- Market metric cards
- Insight cards
- Watchlist cards
- Alert cards
- Chat messages
- Sidebar and topbar
- Loading, empty, and error states

## Motion

Motion should be purposeful:

- Page transitions
- Radar sweep animation
- Pulsing signal points
- Typing indicator
- Smooth score/progress updates

Avoid distracting motion in dense dashboard contexts.
